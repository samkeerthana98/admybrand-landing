-  Dark/Light mode ready (optional)

##  Tech Stack

- **Framework**: Next.js 14 (App Router + TypeScript)
- **Styling**: Tailwind CSS, CSS Modules
- **Animations**: Framer Motion
- **Form Handling**: React Hook Form + Zod
- **Icons**: Lucide Icons
